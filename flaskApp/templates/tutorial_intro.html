This is a plug-and-play Flask app that's meant to be a starting point for your own app. The app has code written to do the following:

  • Authenticate a user on Venmo with the server-side OAuth flow
  • Make a real payment on the Venmo api by issuing a HTTP POST request
  • Get a list of the user's recent transactions by issuing a HTTP GET request
  • Keep track of an authenticated user via sessions
